This guide addresses frequently encountered problems and provides step-by-step solutions.

## Common Issues

### 1. File Not Found

If you encounter a "File Not Found" error, ensure that the file path is correct and that the file exists in the specified location. Check for typos in the file name or path.

### 2. No Adapters Detected

If no adapters are being detected within a configuration, check that your configuration elements use the correct capitalization as required by the **FrankFramework**.

**Correct**
```xml
<Adapter>
...
</Adapter>
```

**Incorrect**
```
<adapter>
...
</adapter>
```

This is required for all configuration elements, and not just limited to adapters alone.
